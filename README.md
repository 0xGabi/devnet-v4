# Running a node (Lighthouse/Geth) on EIP-4844 devnet-4

Docker Compose setup for running a Prysm/Geth node on devnet v4. Uses docker images from `eip4844` branches of Prysm and Geth.

## Run with Docker

```
docker-compose up -d
```
