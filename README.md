# Running a node (Lighthouse/Geth) on EIP-4844 devnet-5

Docker Compose setup for running a Lighthouse/Geth node on devnet v5. Builds docker images from `eip4844` branches of Lighthouse and Geth.

## Run with Docker

```
docker-compose up -d
```
